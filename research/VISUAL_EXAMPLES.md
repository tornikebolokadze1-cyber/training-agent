## Animation Examples (CSS + Timing)

### Streak Update Animation
```
@keyframes streakUpdate {
  0% {
    transform: scale(1);
    opacity: 1;
  }
  50% {
    transform: scale(1.15);
    opacity: 1;
  }
  100% {
    transform: scale(1);
    opacity: 1;
  }
}

.streak-widget.updated {
  animation: streakUpdate 0.5s cubic-bezier(0.34, 1.56, 0.64, 1);
}

/* Trigger: */
/* User completes session → API returns success →
   Add 'updated' class → Animation plays → Remove class after 500ms */
```

### Badge Unlock Animation
```
@keyframes badgeUnlock {
  0% {
    transform: scale(0) rotate(0deg);
    opacity: 0;
  }
  50% {
    transform: scale(1.2) rotate(180deg);
  }
  100% {
    transform: scale(1) rotate(360deg);
    opacity: 1;
  }
}

@keyframes badgeGlow {
  0%, 100% {
    box-shadow: 0 0 8px rgba(255, 215, 0, 0.4);
  }
  50% {
    box-shadow: 0 0 20px rgba(255, 215, 0, 0.8);
  }
}

.achievement-badge.newly-unlocked {
  animation: badgeUnlock 0.8s cubic-bezier(0.34, 1.56, 0.64, 1);
}

.achievement-badge.newly-unlocked {
  animation: badgeGlow 2s ease-in-out infinite;
}

Timing: Unlock animation plays once, glow continues for 3 seconds
```

### XP Bar Fill Animation
```
@keyframes xpFill {
  from {
    width: var(--old-width, 0%);
  }
  to {
    width: var(--new-width, 100%);
  }
}

.xp-bar-fill.gaining {
  animation: xpFill 0.8s cubic-bezier(0.25, 0.46, 0.45, 0.94);
}

/* + Color shift for visual drama */
.xp-bar-fill {
  background: linear-gradient(
    90deg,
    #FBBF24 0%,
    #F59E0B 50%,
    #EF4444 100%
  );
}

Timing: 800ms smooth fill, color gradient emphasizes progress
```

## Mobile Spacing & Layout Grid

### 8px Grid System (Industry Standard)
```
All measurements in multiples of 8px:
├─ 8px   (1 unit)
├─ 16px  (2 units) ← Most common padding
├─ 24px  (3 units) ← Section spacing
├─ 32px  (4 units)
└─ 40px  (5 units) ← Large gaps

Example card:
┌─────────────────────────────┐
│ 16px padding (all sides)    │ ← 2 units
│ ┌─────────────────────────┐ │
│ │ Metric Title            │ │
│ │                         │ │ ← 8px gap (1 unit)
│ │ 87                      │ │
│ │                         │ │ ← 8px gap (1 unit)
│ │ Updated 2h ago          │ │
│ └─────────────────────────┘ │
│                             │
└─────────────────────────────┘
```

### Touch Target Minimum (iOS/Android)
```
❌ Small: 30×30px (too small, mis-tap)
⚠️  OK: 40×40pt (minimum)
✅ Good: 44×44pt (recommended)
✅ Excellent: 48×48pt (generous, no mis-taps)

Bottom nav icon sizing:
├─ Icon: 24×24px
├─ Container: 48×48pt (3× icon size)
├─ Gap between icons: 16px
└─ Vertical padding: 8px top/bottom (seat icon in nav)
```

## What Makes Each Product's Unique Visual Strength

| Product | Unique Visual | Why It Works |
|---------|--------------|--------------|
| **WHOOP** | 3-color dial system | Instant status without reading |
| **Oura** | Concentric rings | Shows data richness elegantly |
| **Garmin** | Customizable charts | Power users feel in control |
| **Duolingo** | Streak flame 🔥 | Instantly recognizable, emotional |
| **Chess.com** | Rating line graph | Clear progression over time |
| **Stripe** | Sparklines | Trends at-a-glance |
| **Linear** | Kanban board | Status visualization without metrics |
| **Notion** | Customizable widgets | Users build their own dashboard |

**Lesson**: Pick ONE visual signature for your dashboard. It will become iconic.

For a trainer dashboard, I recommend: **The Circular Progress Ring (WHOOP style)** because:
1. Users recognize it instantly
2. No reading required (just colors)
3. Works at any screen size
4. Feels premium/modern
5. Easy to animate on unlock
