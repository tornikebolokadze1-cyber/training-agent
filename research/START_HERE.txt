KEY FINDINGS (The 5 Golden Rules)
================================================================================

RULE 1: The Hero Metric
→ Pick ONE metric that dominates your dashboard
→ Display it 40pt+, color-coded ring (not text)
→ For trainer: Pick sessions/streak/XP level

RULE 2: The 3-Color Status System
→ GREEN (#10B981): Good (70-100%)
→ YELLOW (#F59E0B): Caution (40-69%)
→ RED (#EF4444): Alert (0-39%)
→ No reading required - instant recognition

RULE 3: Daily Streaks With Weekly Reset
→ Users who maintain 7-day streaks: 3.6x higher retention
→ CRITICAL: Include freeze feature or weekly reset
→ Prevents streak anxiety (WHOOP's mistake)

RULE 4: Visible Progress & Unlocks
→ Animated progress bars → satisfaction
→ Achievement badges (4 rarity tiers) → completionism
→ Milestones celebrations → dopamine

RULE 5: Mobile-First Everything
→ 90% of users access dashboards on phones
→ Bottom navigation (not top)
→ 44×44pt touch targets minimum
→ Vertical card stack (not tables/grids)
