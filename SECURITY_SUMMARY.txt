================================================================================
TOP 3 FIXES TO PRIORITIZE
================================================================================

1. CRITICAL: Google OAuth Refresh Validation (Railway)
   Issue: On Railway, refresh can fail silently without raising error
   Impact: Could cause auth failures during long-lived sessions
   Fix: Add explicit try/catch with validation before returning creds
   Files: tools/integrations/gdrive_manager.py (lines 65-76)
   Effort: 5 minutes

2. IMPORTANT: Zoom Timestamp Validation
   Issue: Current code accepts timestamps up to 5 min in FUTURE
   Impact: Opens 5-minute window for replay attacks
   Risk: Low (Zoom controls message body, hard to forge), but fixable
   Fix: Tighten to past-only with clock skew tolerance
   Files: tools/app/server.py (lines 560-566)
   Effort: 5 minutes

3. IMPORTANT: WhatsApp Message Deduplication
   Issue: No sender-based dedup; could process same message twice
   Impact: Duplicate AI responses, confused assistant state
   Risk: Medium (could affect user experience)
   Fix: Add message cache keyed by sender_id + message_id
   Files: tools/app/server.py (lines 453-514)
   Effort: 20 minutes

================================================================================
KNOWN LIMITATIONS & TRADE-OFFS
================================================================================

1. WhatsApp Green API requires token in URL
   → Due to Green API protocol design (doesn't support Authorization header)
   → Mitigation: Use HTTPS + restricted network access

2. Rate limiting is IP-based on Railway
   → Safe because Railway + Cloudflare validate headers
   → Would need request signing for untrusted proxies

3. Google OAuth scope is broad (full Drive access)
   → Google Drive doesn't offer granular scopes for service accounts
   → Mitigation: Use service account (not user account) when possible

4. Pinecone API key cannot be rotated without restarting
   → API key is cached; no refresh mechanism exists
   → Mitigation: Add periodic validation (included in fixes)
