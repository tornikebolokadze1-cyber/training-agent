# Commit and Push

> ქართულად: **Commit და Push**

## აღწერა

"Commit" არის ცვლილებების ლოკალურად შენახვა Git რეპოზიტორიაში, რაც სამუშაოს ერთ კონკრეტულ ვერსიას აფიქსირებს. "Push" კი ნიშნავს ამ ლოკალური ცვლილებების გაგზავნას დისტანციურ (მაგ. GitHub-ის) რეპოზიტორიაში, რაც უზრუნველყოფს სარეზერვო ასლს და საშუალებას აძლევს სხვა თანამშრომლებს მიიღონ უახლესი განახლებები.
