# Project Initialization Workflow

> ქართულად: **პროექტის ინიციალიზაციის სამუშაო პროცესი**

## აღწერა

ეს არის სამეტაპიანი პროცესი VS Code-ში AI პროექტის სწორად ასაწყობად: საბაზისო ინფრასტრუქტურის მოწყობა GitHub შაბლონის გამოყენებით, შემდეგ `/setup` ბრძანების გაშვება პროექტის გარემოს ავტომატური კონფიგურაციისთვის და ბოლოს, საჭირო უნარებისა (Skills) და ხელსაწყოების (Tools) ინსტალაცია Claude Code-ის მეშვეობით. ეს უზრუნველყოფს მრავალჯერადი გამოყენების, სტანდარტიზებულ პროექტის სტრუქტურას ყველა მომავალი პროექტისთვის.
