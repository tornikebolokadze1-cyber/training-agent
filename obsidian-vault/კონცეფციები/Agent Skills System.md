# Agent Skills System

> ქართულად: **აგენტის უნარების სისტემა**

## აღწერა

სტრუქტურირებული მიდგომა, სადაც AI აგენტს ენიჭება კონკრეტული, დასახელებული შესაძლებლობები დავალებების უკეთესი მარშრუტიზაციისა (routing) და სწორი ქვე-სისტემის გამოძახებისთვის. ეს სისტემა გაამარტივებს კომპლექსური მულტი-ფუნქციური აგენტების მართვას.
