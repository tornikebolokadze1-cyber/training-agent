# Global Configuration

> ქართულად: **გლობალური კონფიგურაცია**

## აღწერა

ინსტრუმენტის დაყენების მიდგომა, სადაც პარამეტრები ერთხელ იწერება სისტემის დონეზე და მოქმედებს ყველა პროექტში უნივერსალურად. ეს მეთოდი განსხვავდება პროექტის დონის კონფიგურაციისგან, სადაც ყოველ ახალ პროექტში ინსტრუმენტის ხელახლა კონფიგურაცია სახდება.
