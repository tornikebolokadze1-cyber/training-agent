# Environment Variables

> ქართულად: **გარემოს ცვლადები (.env ფაილები)**

## აღწერა

უსაფრთხოების კრიტიკული პრაქტიკა — API გასაღებების შენახვა პროექტის .env ფაილში, ჩატის ფანჯარაში ჩასმის ნაცვლად. ლექციაზე ხაზი გაესვა, რომ API გასაღებების ჩატში გაზიარება კატეგორიულად დაუშვებელია, რადგან ეს უსაფრთხოების სერიოზული დარღვევაა.
