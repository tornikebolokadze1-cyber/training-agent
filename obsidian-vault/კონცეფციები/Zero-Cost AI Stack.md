# Zero-Cost AI Stack

> ქართულად: **ნულოვანი დანახარჯის AI სტეკი**

## აღწერა

n8n (ლოკალური) + Claude Code (ინსტალაციისთვის) + ლოკალური LLM-ის კომბინაცია, რომელიც ქმნის სრულ AI ავტომატიზაციის გარემოს ნულოვანი ან მინიმალური ოპერაციული ხარჯებით. ეს მიდგომა განსაკუთრებით ღირებულია ქართული მცირე და საშუალო ბიზნესებისთვის.
