# Prompt Types

> ქართულად: **პრომპტების ტიპები**

## აღწერა

ლექციაში გამოიყო სამი ძირითადი ტიპი: მომხმარებლის პრომპტი (ჩატში ნაწერი), სისტემური პრომპტი (AI კომპანიის 30-35 გვერდიანი ინსტრუქციები) და მორგებული ინსტრუქციები (მომხმარებლის შექმნილი სისტემური ინსტრუქციები). ამ სამ დონეს შორის განსხვავების გაგება AI-ის ეფექტური გამოყენების ფუნდამენტია.
