# Web Scraping

> ქართულად: **ვებ-სკრეპინგი**

## აღწერა

მონაცემების აღების პროცესი ვებ-საიტებიდან. ლექციაში დაფარულია 3 მეთოდი: პირდაპირი ინტეგრაცია (API), Playwright MCP (ბრაუზერზე დაფუძნებული), და მესამე მხარის სკრეპინგ პლატფორმები (Firecrawl, Epiphy, Cloudflare). თითოეულს აქვს სხვადასხვა რისკი და ხარჯი.
