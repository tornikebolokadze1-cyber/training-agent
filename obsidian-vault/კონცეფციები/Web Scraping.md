# Web Scraping

> ქართულად: **ვებ-სკრეპინგი**

## აღწერა

ვებ-გვერდებიდან მონაცემების ავტომატური შეგროვების პროცესი, რომელიც ლექციაზე განხილული იყო სამი მეთოდის კონტექსტში: პირდაპირი API ინტეგრაცია, Playwright MCP-ზე დაფუძნებული ბრაუზერული ავტომატიზაცია და მესამე მხარის სპეციალიზებული სკრეპინგ პლატფორმები. ყველაზე ეფექტურ მეთოდად დასახელდა პირდაპირი ინტეგრაცია, თუმცა მასთან დაკავშირებულია მონაცემთა კონფიდენციალურობის რისკი.
