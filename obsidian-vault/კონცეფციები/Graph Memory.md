# Graph Memory

> ქართულად: **გრაფული მეხსიერება**

## აღწერა

ვექტორულზე უფრო განვითარებული მეხსიერების ტიპი, რომელიც არა მხოლოდ იძებნის შესაბამის ინფორმაციას, არამედ აკავშირებს და იღებს ინფორმაციებს შორის არსებულ რელაციურ კავშირებს. მაგალითად, იურიდიული დოკუმენტში მე-2 პუნქტის მოძებნისას ის ასევე აბრუნებს მე-11 პუნქტს, თუ ისინი რელაციურად დაკავშირებულია.
