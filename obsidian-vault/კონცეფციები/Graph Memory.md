# Graph Memory

> ქართულად: **გრაფული მეხსიერება**

## აღწერა

ვექტორულ მეხსიერებაზე უფრო განვითარებული ტიპი, რომელიც არა მხოლოდ პოულობს შესაბამის ინფორმაციას, არამედ ავლენს და ინახავს ინფორმაციებს შორის არსებულ რელაციურ კავშირებს. მაგალითად, იურიდიულ კონტრაქტში მე-2 და მე-11 პუნქტებს შორის კავშირის ამოცნობა და ამოღება ერთდროულად.
