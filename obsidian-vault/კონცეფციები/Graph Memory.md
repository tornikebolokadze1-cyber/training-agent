# Graph Memory

> ქართულად: **გრაფული მეხსიერება**

## აღწერა

ვექტორულ მეხსიერებაზე განვითარებული ტიპი, რომელიც არა მხოლოდ ინფორმაციას იძიებს, არამედ ესმის ინფორმაციებს შორის არსებული რელაციური კავშირები. მაგალითად, იურიდიულ კონტრაქტში მე-2 პუნქტსა და მე-11 პუნქტს შორის კავშირის ამოცნობა და ერთდროულად ამოღება.
