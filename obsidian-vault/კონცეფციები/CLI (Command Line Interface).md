# CLI (Command Line Interface)

## აღწერა

ბრძანების ხაზის ინტერფეისი არის ტექსტზე დაფუძნებული ინტერფეისი ოპერაციული სისტემასთან ან პროგრამასთან ურთიერთობისთვის. ის კიდევ ერთი დამაკავშირებელი ინსტრუმენტია, რომელსაც MCP-ზე მეტი მოქმედების შესრულება შეუძლია და ხშირად უფრო სწრაფი, იაფი და ეფექტურია კონკრეტული ამოცანებისთვის.
