# CLI (Command Line Interface)

## აღწერა

CLI (Command Line Interface) არის ტექსტზე დაფუძნებული ინტერფეისი, რომლის საშუალებითაც მომხმარებელს შეუძლია კომპიუტერთან ურთიერთობა ბრძანებების შეყვანით, ნაცვლად გრაფიკული მომხმარებლის ინტერფეისისა (GUI). AI აგენტებს შეუძლიათ გამოიყენონ CLI ინსტრუმენტები, რათა შეასრულონ მოქმედებები ოპერაციულ სისტემაზე ან ურთიერთქმედონ ვებ-სერვისებთან უფრო ეფექტურად, ვიდრე ზოგიერთი სხვა პროტოკოლით.
