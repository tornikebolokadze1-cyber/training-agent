# Project-Level Configuration

> ქართულად: **პროექტის დონის კონფიგურაცია**

## აღწერა

კონფიგურაციის მეთოდი, სადაც პარამეტრები მითითებულია ცალკეული პროექტის ფარგლებში. ამ შემთხვევაში, ყოველ ახალ პროექტში ინსტრუმენტი აღდგება და კონფიგურირდება დაგებული. გლობალური კონფიგურაციის შედარებით, ეს მეთოდი უფრო დროდატი და პიროვნული.
