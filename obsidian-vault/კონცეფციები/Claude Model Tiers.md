# Claude Model Tiers

> ქართულად: **Claude მოდელების იერარქია**

## აღწერა

Anthropic-ის სხვადასხვა Claude მოდელები — Haiku, Sonnet და Opus — განსხვავდებიან სიჩქარის, ღირებულებისა და შესაძლებლობების მიხედვით. სესიაში Sonnet-დან Opus-ზე გადასვლა ტაქტიკური ნაბიჯი იყო, მაგრამ სტრატეგიული სწავლება — როდის გამოვიყენოთ რომელი მოდელი — გამოტოვებულ იქნა.
