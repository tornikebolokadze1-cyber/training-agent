# Smart Script Generation

> ქართულად: **ჭკვიანი სკრიპტის გენერაცია**

## აღწერა

Magic Light AI-ის ფუნქცია, სადაც მომხმარებელი მიაწოდებს მოკლე შინაარსს ან პრომპტს და AI თავად ყოფს მას სცენებად, მინიჭებს ვიზუალებს და ავტომატურად აგენერირებს სრულ ანიმაციას. ეს მიდგომა Basic Script-ისაგან განსხვავდება — Basic Script-ში მომხმარებელი სცენებს ხელით ყოფს, რაც მეტ კონტროლს იძლევა.
