# Computer Use (AI browser control)

> ქართულად: **კომპიუტერის გამოყენება (AI ბრაუზერის მართვა)**

## აღწერა

კომპიუტერის გამოყენება არის სარეზერვო მეთოდი AI აგენტების გარე პლატფორმებთან დასაკავშირებლად, როდესაც პირდაპირი API წვდომა მიუწვდომელია. ამ დროს AI რობოტულად მართავს ბრაუზერს, ხსნის ვებსაიტებს და ასრულებს მოქმედებებს ისე, როგორც ამას ადამიანი გააკეთებდა, თუმცა ეს ნაკლებად ოპტიმალური შემოვლითი გზაა.
