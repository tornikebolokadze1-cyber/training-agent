# Computer Use

> ქართულად: **კომპიუტერის გამოყენება**

## აღწერა

AI-ს შესაძლებლობა, ავტონომიურად გააკონტროლოს ბრაუზერი ან სხვა კომპიუტერული ინტერფეისი, ეკრანზე ვიზუალური ელემენტების ამოცნობისა და ურთიერთქმედების გზით. ლექციაზე დემონსტრირებული იქნა Playwright MCP-ის მეშვეობით, როდესაც Claude-მა გახსნა LinkedIn და ავტონომიურად შეაგროვა ინფორმაცია.
