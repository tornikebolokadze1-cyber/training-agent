# Computer Use

## აღწერა

Computer Use არის შემოვლითი ტექნიკა (workaround), რომელიც გამოიყენება მაშინ, როდესაც AI მოდელს არ აქვს პირდაპირი API წვდომა რაიმე სერვისთან. ამ მეთოდის დროს, ხელოვნური ინტელექტი ფიზიკურად ნავიგაციას უწევს ბრაუზერის ინტერფეისებს და ურთიერთქმედებს ვებგვერდებთან ისევე, როგორც ადამიანი, რაც საშუალებას აძლევს, შეასრულოს ისეთი ამოცანები, როგორიცაა მონაცემთა შეგროვება ან ფორმების შევსება.
