# Computer Use

> ქართულად: **კომპიუტერის გამოყენება**

## აღწერა

Claude-ის შესაძლებლობა, რომელიც AI-ს საშუალებას აძლევს დამოუკიდებლად გააკონტროლოს კომპიუტერი — დააჭიროს ღილაკებს, გახსნას პროგრამები და შეასრულოს ამოცანები. სესიაში ჩართულ იქნა ახსნის გარეშე, მიუხედავად იმისა, რომ შეიცავს მნიშვნელოვან რისკებს, მათ შორის შემთხვევით მონაცემების წაშლის ან არასასურველი მოქმედებების განხორციელების შესაძლებლობას.
