# Computer Use

> ქართულად: **კომპიუტერის გამოყენება (ბრაუზერის ავტომატიზაცია)**

## აღწერა

ტექნოლოგია, რომელიც AI აგენტს საშუალებას აძლევს ავტონომიურად გახსნას ბრაუზერი, გადავიდეს ვებსაიტებზე და შეასრულოს მოქმედებები API წვდომის გარეშე. ლექტორმა ეს აღწერა, როგორც API-ს 'შემოვლითი გზა' და Manus AI-ის ერთ-ერთ მთავარ შესაძლებლობად წარმოადგინა.
