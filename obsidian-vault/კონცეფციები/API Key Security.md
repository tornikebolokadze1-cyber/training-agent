# API Key Security

> ქართულად: **API გასაღების უსაფრთხოება**

## აღწერა

API გასაღებების (credentials) სწორი და უსაფრთხო მართვის პრაქტიკა, რომლის კრიტიკული წესია: API გასაღებები არასოდეს უნდა შეიყვანოთ ჩატის ინტერფეისში, მხოლოდ ინსტრუმენტის უსაფრთხო პარამეტრების პანელში. ჩატში შეყვანის შემთხვევაში, არსებობს მონაცემთა გაჟონვის რეალური რისკი.
