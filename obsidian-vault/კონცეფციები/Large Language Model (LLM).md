# Large Language Model (LLM)

> ქართულად: **დიდი ენობრივი მოდელი (LLM)**

## აღწერა

LLM არის ხელოვნური ინტელექტის მოდელი, რომელიც გაწვრთნილია უზარმაზარ ტექსტურ მონაცემთა ნაკრებზე და შეუძლია ბუნებრივი ენის გაგება, გენერირება და დამუშავება. ლექციაზე მას მოიხსენიებენ როგორც "ტვინს" AI აგენტებისთვის და ხაზგასმულია უფასო ან ხელმისაწვდომი ალტერნატივების შერჩევის მნიშვნელობა API-ის მაღალი საფასურის თავიდან ასაცილებლად.
