# Commander Agent Pattern

> ქართულად: **მეთაური-აგენტის პატერნი**

## აღწერა

არქიტექტურული მიდგომა, სადაც ერთი მთავარი ორკესტრატორი აგენტი დინამიურად ქმნის და მართავს ქვე-აგენტებს დავალების საჭიროებიდან გამომდინარე. ეს მიდგომა ბევრად უფრო მასშტაბირებადია, ვიდრე ხელით განსაზღვრული მულტი-აგენტური სისტემები.
