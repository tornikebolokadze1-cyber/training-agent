# Prompt Intermediary Workflow

> ქართულად: **პრომპტის შუამავლის სამუშაო პროცესი**

## აღწერა

მეთოდოლოგია, სადაც ChatGPT გამოიყენება შუამავლად ვიდეო გენერატორებისთვის პრომპტების შესაქმნელად — მომხმარებელი აღწერს სასურველ ვიდეოს და რეფერენს ფოტოებს, ChatGPT კი ქმნის ოპტიმიზებულ პრომპტს კონკრეტული გენერატორისთვის. ეს მიდგომა გამორიცხავს ხელით პრომპტის წერას და ითვალისწინებს თითოეული პლატფორმის უნიკალურ სინტაქსს.
