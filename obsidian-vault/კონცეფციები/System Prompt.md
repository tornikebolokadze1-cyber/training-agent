# System Prompt

> ქართულად: **სისტემური პრომფტი**

## აღწერა

სისტემური პრომფტი არის AI მოდელისთვის წინასწარ განსაზღვრული ინსტრუქციების ნაკრები, რომელიც მოდელის ქცევას, ტონს და მიდგომას განსაზღვრავს მთელი სესიის განმავლობაში. მაგალითად, Claude-ს შეიძლება ეთქვას, რომ ყოველთვის მიიღოს კრიტიკული, სოკრატული პოზიცია — ყველა წინადადებაზე ავტომატური თანხმობის ნაცვლად.
