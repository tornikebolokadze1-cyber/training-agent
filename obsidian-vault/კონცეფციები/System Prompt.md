# System Prompt

> ქართულად: **სისტემური პრომპტები**

## აღწერა

AI კომპანიის მიერ განსაზღვრული ფარული დოკუმენტი (დაახლოებით 30-35 გვერდი), რომელიც მართავს მოდელის ქცევას. User Prompt-ისა და System Prompt-ის კომბინაცია გავლენას ახდენს AI-ის პასუხებზე.
