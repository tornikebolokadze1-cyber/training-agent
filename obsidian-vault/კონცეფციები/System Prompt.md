# System Prompt

> ქართულად: **სისტემური პრომპტი**

## აღწერა

AI კომპანიის მიერ ჩაშენებული ფარული ინსტრუქციების ნაკრები (30-35 გვერდი), რომელიც განსაზღვრავს მოდელის ქცევას და შეზღუდვებს. მომხმარებელს Custom Instructions-ით შეუძლია ამ ჩარჩოს ფარგლებში თავისი ქცევის ფენის დამატება.
