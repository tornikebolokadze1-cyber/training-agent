# Agent Swarm

> ქართულად: **აგენტების გუნდი**

## აღწერა

არქიტექტურა, სადაც მრავალი სპეციალიზებული AI ქვე-აგენტი მუშაობს პარალელურ რეჟიმში კომპლექსური დავალების სხვადასხვა ასპექტზე ერთდროულად. Kimi 2.5-ის Agent Swarm-ის დემოში სხვადასხვა სახელდებული აგენტები (ანალიტიკოსი, მკვლევარი, ინდუსტრიის სპეციალისტი) ერთობლივად ამუშავებდნენ კვლევით დავალებას.
