# Agent Swarm

> ქართულად: **აგენტების გუნდი**

## აღწერა

მრავალი AI ქვე-აგენტი, რომლებიც პარალელურად მუშაობენ კომპლექსური დავალების სხვადასხვა ასპექტზე. კიმი 2.5-ში მიმდინარე დემოშია შეჩვენებული, სადაც თითოეულ აგენტს უნიკალური როლი (მკვლევარი, ანალიტიკოსი) აქვს.
