# Text Formatting for TTS

> ქართულად: **ტექსტის ფორმატირება TTS-ისთვის**

## აღწერა

TTS სისტემების საუკეთესო შედეგის მიღებისთვის ტექსტის სპეციალური მომზადების პრაქტიკა, მათ შორის რიცხვების სიტყვიერი ჩაწერა და პაუზებისთვის სასვენი ნიშნების სწორად გამოყენება. Cartesia-ს დემონსტრაციაში ნაჩვენები იყო, რომ 'ორასზე მეტი' ბევრად უკეთ გამოითქმება, ვიდრე '200+'.
