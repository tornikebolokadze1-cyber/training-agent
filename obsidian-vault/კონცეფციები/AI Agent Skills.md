# AI Agent Skills

> ქართულად: **უნარები (Skills)**

## აღწერა

ეს არის სპეციალიზებული შესაძლებლობები ან ფუნქციები (მაგალითად, Frontend, SEO, Excel Analytics), რომლებიც ემატება Claude Code-ს ან სხვა AI აგენტებს. ისინი ზრდიან აგენტის ეფექტურობას კონკრეტულ ამოცანებში, რაც მას საშუალებას აძლევს შეასრულოს კომპლექსური დავალებები, ისევე როგორც ოსტატს შესაბამისი ინსტრუმენტები.
