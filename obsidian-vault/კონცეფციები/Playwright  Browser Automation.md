# Playwright / Browser Automation

> ქართულად: **Playwright / ბრაუზერის ავტომატიზაცია**

## აღწერა

მონაცემთა მოპოვების "ხულიგნური" მეთოდი, რომლის დროსაც AI ფიზიკურად ხსნის ბრაუზერს (მაგალითად, Chrome-ს), ახორციელებს ნავიგაციას ვებსაიტებზე, იღებს სქრინშოტებს და აგროვებს მონაცემებს. ის გამოიყენება მაშინ, როცა პირდაპირი API წვდომა მიუწვდომელია.
