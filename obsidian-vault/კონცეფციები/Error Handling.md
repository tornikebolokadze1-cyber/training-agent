# Error Handling

> ქართულად: **შეცდომების დამუშავება**

## აღწერა

Claude-ის უკმარი ან არასწორი მოპასუხე შეცდომის გაჩენის შემთხვევაში, უფრო მძლავრი მოდელზე გადართვა (მაგ. Sonnet-დან Opus-ზე) ან პრომპტის გამოსწორება. სესიაში ლიკას შეცდომები დაფიქსირდა და Opus-ზე გადაკვეთილი იქნა პრობლემის გადასაჭრელად.
