# Machine Learning (ML)

> ქართულად: **მანქანური სწავლება (ML)**

## აღწერა

მანქანური სწავლება არის ხელოვნური ინტელექტის ქვედარგი, რომელიც სისტემებს აძლევს უნარს, ავტომატურად ისწავლონ მონაცემებიდან გამოკვეთილი პროგრამირების გარეშე. ის მოიცავს სხვადასხვა მეთოდს, როგორიცაა ზედამხედველობითი, ზედამხედველობის გარეშე და განმტკიცებითი სწავლება, რათა მოდელებმა ისწავლონ პატერნები და მიიღონ გადაწყვეტილებები.
