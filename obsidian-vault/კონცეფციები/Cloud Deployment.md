# Cloud Deployment

> ქართულად: **ქლაუდ დეპლოიმენტი**

## აღწერა

ღრუბლოვანი განთავსება გულისხმობს პროგრამული უზრუნველყოფის გაშვებას და მონაცემების შენახვას მესამე მხარის პროვაიდერის სერვერებზე ინტერნეტის საშუალებით. მიუხედავად იმისა, რომ ის გვთავაზობს მოხერხებულობას და მართულ სერვისებს, მას თან ახლავს ყოველთვიური ხარჯები და მონაცემთა კონფიდენციალურობის შეშფოთება, რადგან მონაცემები ტოვებს მომხმარებლის კონტროლს.
