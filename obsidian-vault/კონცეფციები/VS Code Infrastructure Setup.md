# VS Code Infrastructure Setup

> ქართულად: **VS Code-ის ინფრასტრუქტურის გამართვა**

## აღწერა

ეს არის პროცესი, რომელიც მოიცავს საჭირო პროგრამული უზრუნველყოფის, როგორიცაა GitHub ექსტენცია, Playwright CLI და ბრაუზერი, დაყენებას VS Code-ში. მისი მიზანია AI კოდირების სრულყოფილი სამუშაო გარემოს მომზადება. Claude Code-ს შეუძლია ამ ინფრასტრუქტურის ნულიდან აწყობა პროექტის სახელის მითითებით, თუმცა ხელით კონფიგურაციაც ხშირად საჭიროა.
