# Chatbot with Semantic Memory

## აღწერა

AI ჩატბოტი, რომელსაც აქვს ძლიერი სემანტიკური მეხსიერება და თვეების განმავლობაში შენარჩუნებული სესიის მეხსიერება მომხმარებლის პროფილებზე მიბმული. ბიზნეს გამოყენების კონტექსტში შეიძლება ჩაიდოს ფოტოს ამოცნობა, ფასდაკლებიანი follow-up ემეილების გაგზავნა და პროდუქტის ლინკების გაზიარება.
