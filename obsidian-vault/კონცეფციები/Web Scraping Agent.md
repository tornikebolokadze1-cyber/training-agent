# Web Scraping Agent

> ქართულად: **ვებ სკრეპინგის აგენტი**

## აღწერა

Facebook ჯგუფების კონტენტის ავტომატური მოპოვება Cookie-ებზე დაფუძნებული ავთენტიფიკაციით ბაზრის კვლევის მიზნებისთვის. Cookie-ები ვალიდურია 30 დღე და საჭიროებს ხელით განახლებას — სისტემა ათვალიერებს დახურულ ჯგუფებს, აანალიზებს კონტენტს და აგზავნის შეტყობინებებს.
