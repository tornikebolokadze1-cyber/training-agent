# Context Window

> ქართულად: **კონტექსტის ფანჯარა**

## აღწერა

AI მოდელის 'სამუშაო მეხსიერება', რომელიც განსაზღვრავს, რამდენი ინფორმაცია შეიძლება ერთდროულად დამუშავდეს. ლექტორმა ახსნა ანალოგიით: თანამშრომელი, რომელიც 3 საათი კარგად მუშაობს, შემდეგ კი სჭირდება 'გადატვირთვა'; ქართულად წერა კონტექსტს 2-3-ჯერ სწრაფად ავსებს ტოკენიზაციის სპეციფიკის გამო.
