# Context Window

> ქართულად: **კონტექსტის ფანჯარა**

## აღწერა

Claude Code-ის შეზღუდული და ძვირფასი რესურსი, რომელიც განსაზღვრავს რამდენი ინფორმაცია შეიძლება ერთდროულად გადაამუშავოს მოდელმა. გადატვირთული კონტექსტი აუქმებს Claude Code-ის გამოსადეგობას, ამიტომ სქილების შერჩევითი ინსტალაცია კრიტიკულია.
