# Agent Memory

> ქართულად: **აგენტის მეხსიერება**

## აღწერა

AI აგენტის უნარი შეინარჩუნოს ინფორმაცია სხვადასხვა სესიებს შორის, მათ შორის სემანტიკური და სესიის მეხსიერება. სწორად კონფიგურირებული მეხსიერება საშუალებას აძლევს აგენტს გახსოვდეს, სად გაჩერდა მუშაობა და ისწავლოს ყოველი სესიისგან.
