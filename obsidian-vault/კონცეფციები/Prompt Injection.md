# Prompt Injection

> ქართულად: **პრომპტ ინჟექცია**

## აღწერა

უსაფრთხოების შეტევის სახეობა, სადაც მავნე ინსტრუქციები ჩართულია მონაცემებში, რათა AI-მ არასანქცირებული მოქმედება შეასრულოს. ლექტორის მიხედვით, მონაცემთა კონტროლის ჩართულობის შემთხვევაში გამოცდილ მომხმარებელს თეორიულად შეუძლია სხვის კონფიდენციალური ინფორმაციის ამოღება.
