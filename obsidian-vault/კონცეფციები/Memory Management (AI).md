# Memory Management (AI)

> ქართულად: **მეხსიერების მართვა**

## აღწერა

ხელოვნური ინტელექტის მოდელების ფუნქცია, რომელიც ინახავს მომხმარებლის ინფორმაციას წინა საუბრებიდან ან მორგებული პარამეტრებიდან. ამ მეხსიერების გაფილტვრა, მართვა და წაშლა შესაძლებელია მოდელის მუშაობის ოპტიმიზაციისთვის, რათა თავიდან იქნას აცილებული კონტექსტის გადატვირთვა.
