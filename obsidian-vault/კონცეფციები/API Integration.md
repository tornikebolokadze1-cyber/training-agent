# API Integration

> ქართულად: **API ინტეგრაცია**

## აღწერა

API (Application Programming Interface) არის ოფიციალური, სტრუქტურირებული, კოდური ინტერფეისი პლატფორმებთან დასაკავშირებლად. ყველაზე საიმედო მეთოდია, მაგრამ ტექნიკურად ყველაზე რთული — და თუ API ცუდადაა დაწერილი, ინტეგრაციები ირღვევა.
