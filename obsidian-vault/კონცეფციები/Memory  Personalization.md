# Memory / Personalization

> ქართულად: **მეხსიერება / პერსონალიზაცია**

## აღწერა

AI პლატფორმებში ჩაშენებული ფუნქცია, რომელიც მოდელს საშუალებას აძლევს დაიმახსოვროს მომხმარებლის კონტექსტი, პრეფერენციები და წინა საუბრები სესიებს შორის. ლექტორი ამ ფუნქციის ჩართვას გირჩევს, განსხვავებით მონაცემთა კონტროლის პარამეტრისგან.
