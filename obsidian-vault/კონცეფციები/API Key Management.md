# API Key Management

> ქართულად: **API გასაღების მართვა**

## აღწერა

კრიტიკული უსაფრთხოების პრაქტიკა, რომელიც გულისხმობს API გასაღებების (როგორიცაა Claude API, Google AI Studio API) უსაფრთხოდ შენახვას და გამოყენებას. API გასაღებები აბსოლუტურად არასოდეს უნდა გაკოპირდეს ჩატის ინტერფეისში, რადგან მოკიდებულია მონაცემთა გაჟონვის რისკი.
