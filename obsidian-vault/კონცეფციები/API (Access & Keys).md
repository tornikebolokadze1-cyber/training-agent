# API (Access & Keys)

> ქართულად: **API გასაღებები**

## აღწერა

უნიკალური კოდები, რომლებიც აღსაზღვრავენ წვდომას გარე სერვისებზე (Claude API, ChatGPT API, Firecrawl API). მათი უსაფრთხოება კრიტიკულია - უნდა შეინახოს გარემოს ცვლადებში (.env ფაილები) და არასდროს ღიად კოდში. გაჟონილმა გასაღებმა შეიძლება გამოიწვიოს არასანქცირებული გამოყენება და დიდი ფინანსური ზარალი.
