# API (Access & Keys)

> ქართულად: **API კავშირი**

## აღწერა

API (Application Programming Interface) კავშირი არის AI აგენტების გარე პლატფორმებთან დასაკავშირებლად უპირატესი მეთოდი, რომელიც იყენებს API გასაღებებს ან MCP (Model Context Protocol)-ს პირდაპირი ხიდის შესაქმნელად. ამ მეთოდით აგენტი პირდაპირ ეკონტაქტება პლატფორმის სისტემას ბრაუზერის გახსნის გარეშე, რაც უფრო ეფექტურია.
