# API (Access & Keys)

> ქართულად: **API გასაღებები და მათი უსაფრთხოება**

## აღწერა

გარე სერვისებთან (Claude API, OpenAI API, Firecrawl API) კავშირისთვის საჭირო ავთენტიფიკაციის გასაღებები, რომლებიც აგენტების შექმნისთვის აუცილებელია. ლექტორმა ხაზი გაუსვა კრიტიკულ უსაფრთხოების წესს: გასაღებები უნდა ინახებოდეს გარემოს ცვლადებში (.env ფაილები) და არასდროს გამოჩნდეს ღიად კოდში, რადგან გაჟონვამ შეიძლება გამოიწვიოს მნიშვნელოვანი ფინანსური ზარალი.
