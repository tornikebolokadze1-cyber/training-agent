# MCP (Model Context Protocol)

## აღწერა

MCP არის Anthropic-ის მიერ შემუშავებული სტანდარტული პროტოკოლი, რომელიც AI მოდელებს გარე ინსტრუმენტებთან და სერვისებთან ინტეგრაციის საშუალებას აძლევს. ლექციაში მოხსენიებული MCP ინტეგრაციები მოიცავს N8N-ს, ვებ-სკრაპინგს, GitHub-ს, Figma-ს, Notion-სა და მეხსიერება/RAG სისტემებს.
