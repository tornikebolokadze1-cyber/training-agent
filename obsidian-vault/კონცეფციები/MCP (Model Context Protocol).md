# MCP (Model Context Protocol)

> ქართულად: **მოდელის კონტექსტის პროტოკოლი (MCP)**

## აღწერა

MCP (Model Context Protocol) არის Anthropic-ის ღია სტანდარტი, რომელიც AI მოდელებს საშუალებას აძლევს, დაუკავშირდნენ გარე მონაცემთა წყაროებს და ინსტრუმენტებს. მიუხედავად იმისა, რომ ლექციაზე ის პირდაპირ არ იყო ახსნილი, მისი კონფიგურაცია ("ლინკების" და "კონფიგურაციების" ინსტალაცია) სავარაუდოდ ამ პროტოკოლს ეხებოდა, რაც AI სისტემების ინტეგრაციისთვის კრიტიკული კომპონენტია.
