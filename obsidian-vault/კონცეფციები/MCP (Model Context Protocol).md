# MCP (Model Context Protocol)

> ქართულად: **MCP კავშირი**

## აღწერა

Anthropic-ის მიერ შემუშავებული სტანდარტული პროტოკოლი, რომელიც AI მოდელებს საშუალებას აძლევს გარე ინსტრუმენტებთან და სერვისებთან კომუნიკაციისა. MCP სერვერის კონფიგურაციის მეშვეობით შესაძლებელია Claude Code-სა და N8N-ს შორის ინტეგრაცია, რაც საშუალებას იძლევა ბუნებრივი ენის ბრძანებებით სამუშაო პროცესების მართვისა.
