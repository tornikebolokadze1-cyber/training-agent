# MCP (Model Context Protocol)

> ქართულად: **MCP კავშირი**

## აღწერა

Anthropic-ის მიერ შემუშავებული სტანდარტული პროტოკოლი, რომელიც AI მოდელებს საშუალებას აძლევს გარე ინსტრუმენტებთან და სერვისებთან დაკავშირებისა. N8N-ის Claude Code-თან ინტეგრირება სწორედ MCP სერვერის კონფიგურაციის გამოყენებით ხდება, რაც AI-ს გარე სისტემების მართვის საშუალებას აძლევს.
