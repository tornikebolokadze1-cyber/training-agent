# Style Reference

> ქართულად: **სტილის რეფერენსი**

## აღწერა

Midjourney-ში სტილის გადატანის ტექნიკა (SRIF — Style Reference Image Feature), რომელიც საშუალებას იძლევა კონკრეტული მხატვრული სტილი (მაგ., ბაროკო) გადაიტანოს ახლად გენერირებულ სურათზე. განსხვავდება Image Reference-ისგან, რომელიც ზოგად ვიზუალურ კომპოზიციას იმეორებს.
