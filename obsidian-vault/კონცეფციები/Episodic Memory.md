# Episodic Memory

> ქართულად: **ეპიზოდური მეხსიერება**

## აღწერა

ამჟამად ხელმისაწვდომი ყველაზე მოწინავე მეხსიერების ტიპი, რომელიც იჭერს დროით განზომილებას და ეპიზოდების თანმიმდევრობას. მაგალითად, შეუძლია დააფიქსიროს სტუდენტის სწავლის პროგრესი ლექციიდან ლექციამდე და დააგენერიროს განვითარების ანგარიში.
