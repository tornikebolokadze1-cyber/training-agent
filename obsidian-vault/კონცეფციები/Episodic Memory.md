# Episodic Memory

> ქართულად: **ეპიზოდური მეხსიერება**

## აღწერა

ამჟამად ხელმისაწვდომი ყველაზე ძლიერი მეხსიერების ტიპი, რომელიც იჭერს დროით განზომილებას და ეპიზოდის აღქმას. აგენტი იმახსოვრებს დროის სეკვენციებს, მომხმარებლის სწავლის პროგრესის დინამიკას და კონტექსტუალურ ისტორიას.
