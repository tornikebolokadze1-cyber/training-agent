# Episodic Memory

> ქართულად: **ეპიზოდური მეხსიერება**

## აღწერა

ამჟამად ხელმისაწვდომი ყველაზე ძლიერი მეხსიერების ტიპი, რომელიც იჭერს დროით განზომილებას — ეპიზოდებისა და მოვლენების ქრონოლოგიურ აღქმას. ეს საშუალებას აძლევს აგენტს თვალყური ადევნოს მომხმარებლის პროგრესს დროთა განმავლობაში და დააგენერიროს განვითარების ანალიზი.
