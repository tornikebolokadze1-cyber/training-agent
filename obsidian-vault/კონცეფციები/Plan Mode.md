# Plan Mode

> ქართულად: **გეგმის რეჟიმი**

## აღწერა

ეს არის მეთოდოლოგია AI-სთან მუშაობისას, როდესაც მომხმარებელი იყენებს Claude-ის ვებ-ვერსიას ან მსგავს ინსტრუმენტს საწყისი სტრუქტურის, იდეების და გეგმის ჩამოსაყალიბებლად, სანამ რეალურ შესრულებაზე გადავა. ის გულისხმობს 1-1.5 საათის დათმობას გეგმის დეტალური შემუშავებისთვის AI-ის დახმარებით.
