# Claude Skills

## აღწერა

ინსტრუქციების ნაკრები, რომელიც ეუბნება Claude-ს როგორ მოიქცეს კონკრეტული დავალებების დროს (ხმის ტემბრი, სტილი, ფორმატი). არსებობს Project Skills (ერთი პროექტისთვის) და Global Skills (ყველა პროექტისთვის). შეიძლება თავიდან შეიქმნას ან საზოგადოების ბიბლიოთეკებიდან იმპორტირდეს.

## პრაქტიკული გამოყენება

- ლექციის რეპორტის ინტერაქტიული კითხვა-პასუხის რეჟიმში გენერირების განმეორებადი 'უნარის' შექმნა და შენახვა (ჯგუფი 1, ლექცია 2)
- მაისში თბილისი/ქუთაისიდან ბარსელონაში იაფი ბილეთების მოძიების უნარი შეიქმნა და დაიტესტა Claude Cowork-ში. (ჯგუფი 1, ლექცია 4)
- 'მთის ყავის' ბრენდის ხმის უნარი — Claude ავტომატურად ამ სტილში წერს Facebook, Instagram, LinkedIn კონტენტს, პრესრელიზებს და პროდუქტის აღწერებს. (ჯგუფი 1, ლექცია 4)
- Frontend დიზაინერის, SEO ოპტიმიზატორის ან MCP builder უნარების იმპორტირება საზოგადოების ბიბლიოთეკიდან (ჯგუფი 2, ლექცია 2)
- მარიამის Meta Ads აგენტისთვის რეკლამის აღწერებისა და პრიორიტეტების მართვის უნარის დამატება. (ჯგუფი 2, ლექცია 6)
