# Supervised Learning

> ქართულად: **ზედამხედველობითი სწავლება**

## აღწერა

ეს არის მანქანური სწავლების მეთოდი, სადაც მოდელი სწავლობს მარკირებული (Labeled) მონაცემების ნაკრებიდან, რომელსაც უკვე აქვს სწორი გამოსავალი. მაგალითად, სისტემას ეძლევა კატებისა და ძაღლების ათასობით ფოტო, რომელთაგან თითოეულს აწერია, კატაა თუ ძაღლი, რის საფუძველზეც მოდელი სწავლობს კლასიფიკაციას.
