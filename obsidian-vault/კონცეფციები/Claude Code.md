# Claude Code

## აღწერა

Claude Code არის Anthropic-ის AI მოდელის სპეციალიზებული რეჟიმი, რომელიც ორიენტირებულია კოდის გენერირებაზე, პროექტის სტრუქტურის შექმნაზე და გარე აპლიკაციებთან ინტეგრაციაზე. ის საშუალებას აძლევს მომხმარებლებს ნულიდან, კოდით შექმნან აგენტური პროცესები ან დაუკავშირდნენ არსებულ სისტემებს, რაც მნიშვნელოვნად ცვლის აგენტების განვითარების პარადიგმას.

## პრაქტიკული გამოყენება

- სტუდენტის მიერ შექმნილი აგენტის პროექტის კოდის თვითშეფასება და ხარისხის ანალიზი სხვადასხვა კრიტერიუმით. (ჯგუფი 2, ლექცია 3)
- VS Code-ში პროექტის ინფრასტრუქტურის გამართვა და `/setup` ბრძანების ლაივ დემონსტრაცია. (ჯგუფი 1, ლექცია 5)
- VS Code-ში ახალი პროექტის ნულიდან შექმნა და Claude Code-ის ინიციალიზაცია. (ჯგუფი 1, ლექცია 4)
- Claude-ისთვის დეტალური პრომპტის დაწერა airmpi.com-დან უნარების/ინსტრუმენტების ინტეგრირებისთვის. (ჯგუფი 1, ლექცია 4)
- ტარიელის სასტუმროების მონიტორინგის პროექტის ინიციალიზაცია. (ჯგუფი 2, ლექცია 2)
