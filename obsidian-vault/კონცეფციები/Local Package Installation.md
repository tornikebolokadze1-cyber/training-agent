# Local Package Installation

> ქართულად: **ლოკალური პაკეტის ინსტალაცია**

## აღწერა

ლოკალური პაკეტის ინსტალაცია გულისხმობს პროგრამული დამოკიდებულებების ან ინსტრუმენტების კონკრეტული პროექტის საქაღალდეში დაყენებას, რათა თავიდან იქნას აცილებული კონფლიქტები სხვა პროექტებთან. ლექციაზე ეს კონცეფცია გლობალურ ინსტალაციასთან კონტრასტში იყო განხილული, როგორც პროგრამული გარემოს მართვის ალტერნატიული მეთოდი.
