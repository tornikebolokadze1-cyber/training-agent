# Vector Memory

> ქართულად: **ვექტორული მეხსიერება**

## აღწერა

AI აგენტის მეხსიერების ბაზისური ტიპი, რომელიც ტექსტს ყოფს ნაწილებად (chunks) და ანიჭებს მათ რიცხვით მნიშვნელობებს (embedding). მოთხოვნისას სისტემა იძიებს სემანტიკურად ყველაზე მსგავს ნაწილებს ვექტორულ მონაცემთა ბაზაში.
