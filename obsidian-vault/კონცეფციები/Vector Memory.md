# Vector Memory

> ქართულად: **ვექტორული მეხსიერება**

## აღწერა

AI აგენტის მეხსიერების საბაზისო ტიპი, სადაც ტექსტი იყოფა ნაწილებად (chunks) და თითოეულს ენიჭება რიცხვითი მნიშვნელობა (embedding). მოთხოვნისას სისტემა ეძებს სემანტიკურად ყველაზე მსგავს ნაწილებს და აბრუნებს მათ კონტექსტად.
