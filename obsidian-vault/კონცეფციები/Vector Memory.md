# Vector Memory

> ქართულად: **ვექტორული მეხსიერება**

## აღწერა

საბაზისო მეხსიერების ტიპი, რომელზეც აშენდა თანამედროვე AI სისტემები. აგენტი ტექსტის ნაწილებს (chunks) ემბედინგის მეშვეობით აკეთებს რიცხვით მნიშვნელობებს და ინახავს მათ ვექტორულ მონაცემთა ბაზაში. მოთხოვნისას სემანტიკურად მსგავსი ნაწილები იძებნება ბაზიდან.
