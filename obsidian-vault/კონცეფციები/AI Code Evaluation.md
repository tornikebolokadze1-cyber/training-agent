# AI Code Evaluation

> ქართულად: **AI კოდის შეფასება**

## აღწერა

ხელოვნური ინტელექტის გამოყენების ტექნიკა პროგრამული კოდის ხარისხის, უსაფრთხოების, სტრუქტურისა და პროდაქშენისთვის მზაობის შესაფასებლად. ლექციაზე ნაჩვენები იყო, თუ როგორ შეაფასა Claude Code-მა სტუდენტის აგენტის პროექტი და ქულები მიანიჭა სხვადასხვა კრიტერიუმით. ლექტორმა ასევე ურჩია ChatGPT Codex-ის გამოყენება ჯვარედინი შეფასებისთვის, რაც ხაზს უსვამს AI-ის როლს დეველოპერულ პროცესებში.
