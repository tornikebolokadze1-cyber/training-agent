# Advisor Agent

> ქართულად: **მრჩეველი აგენტი**

## აღწერა

ლექტორის კურსის მართვის პერსონალური AI სისტემა, რომელიც კურსის ფილოსოფიის მთავარ პრაქტიკულ დემონსტრაციად გამოიყენება. აგენტი ინახავს ლექციების ჩანაწერებს, ტრანსკრიფციებს, WhatsApp-ის ჩატის ისტორიებს და ყოველი ლექციის შემდეგ 10-გვერდიან სიღრმისეულ ანგარიშს ამზადებს.
