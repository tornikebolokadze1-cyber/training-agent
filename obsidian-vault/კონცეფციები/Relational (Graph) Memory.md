# Relational (Graph) Memory

> ქართულად: **რელაციური (გრაფ) მეხსიერება**

## აღწერა

მეხსიერების ტიპი, სადაც ინფორმაციის კვანძები ერთმანეთთან დაკავშირებულია, რაც AI-ს საშუალებას აძლევს გაიგოს ფაქტებს შორის ურთიერთდამოკიდებულება. ეს გადასცდება კლასიკურ 'ატვირთე-იკითხე' მოდელს და ნაჩვენებია Obsidian-ის ვიზუალური ცოდნის რუქის მეშვეობით.
