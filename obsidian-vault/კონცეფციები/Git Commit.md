# Git Commit

## აღწერა

`/commit` არის ბრძანება, რომელიც გამოიყენება ვერსიის კონტროლის სისტემებში, როგორიცაა Git, რათა შენახულ იქნას ცვლილებები კოდის საცავში (repository). AI აგენტებს შეუძლიათ ავტომატურად შეასრულონ `commit` ოპერაცია, კოდის შექმნის ან მოდიფიკაციის შემდეგ, რაც აადვილებს პროექტის მართვას და ცვლილებების ისტორიის შენარჩუნებას GitHub-ზე ატვირთვისთვის.
