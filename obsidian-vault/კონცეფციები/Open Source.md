# Open Source

> ქართულად: **ღია კოდის მოდელები**

## აღწერა

ღია კოდი გულისხმობს პროგრამული უზრუნველყოფის კოდს, რომელიც საჯაროდ არის ხელმისაწვდომი, რაც ნებისმიერ მსურველს აძლევს მისი ნახვის, შეცვლისა და გავრცელების უფლებას. ასეთი პროექტები, როგორიცაა GitHub-ზე არსებული საჯარო რეპოზიტორიები, ხელს უწყობს საზოგადოების კონტრიბუციას, გამჭვირვალობას და ინოვაციების დაჩქარებას.
