# LLM

> ქართულად: **დიდი ენობრივი მოდელი**

## აღწერა

LLM (Large Language Model) არის აგენტის 'ტვინი' — ის კომპონენტი, რომელიც ახორციელებს მსჯელობას და გადაწყვეტილების მიღებას. Claude Code-ისგან განსხვავებით, LLM ჩანაცვლებადი კომპონენტია და შეიძლება იყოს Claude API, Gemini, ChatGPT, DeepSeek ან სხვა.
