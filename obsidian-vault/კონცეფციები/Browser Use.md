# Browser Use

> ქართულად: **ბრაუზერის გამოყენება**

## აღწერა

Browser Use არის ინტეგრაციის მეთოდი, რომლის დროსაც AI აგენტი ხსნის Chrome-ს და პირდაპირ ნავიგაციას ახდენს ვებსაიტებზე ოფიციალური API-ის გარეშე. ეს მიდგომა გამოიყენება როგორც Claude-ში, ასევე ChatGPT-ში და განსაკუთრებით სასარგებლოა იმ სერვისებთან, რომლებსაც საჯარო API არ გააჩნიათ.
