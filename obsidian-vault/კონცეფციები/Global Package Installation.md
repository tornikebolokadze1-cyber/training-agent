# Global Package Installation

> ქართულად: **გლობალური პაკეტის ინსტალაცია**

## აღწერა

გლობალური პაკეტის ინსტალაცია გულისხმობს პროგრამული დამოკიდებულებების ან ინსტრუმენტების სისტემის დონეზე დაყენებას, ისე რომ ისინი ხელმისაწვდომი იყოს ნებისმიერი პროექტისთვის. ლექციის დასაწყისში ჯგუფმა დაადასტურა, რომ გარკვეული ინსტრუმენტი სწორედ ასე იყო დაყენებული, რაც პროგრამული გარემოს მართვის მნიშვნელოვანი ასპექტია.
