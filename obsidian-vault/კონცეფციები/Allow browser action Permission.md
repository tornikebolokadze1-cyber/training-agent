# "Allow browser action" Permission

> ქართულად: **"Allow browser action" ნებართვა**

## აღწერა

"Allow browser action" არის ნებართვა Claude-ის სამაგიდო აპლიკაციაში, რომელიც მოდელს საშუალებას აძლევს, დამოუკიდებლად იმოქმედოს ვებ-ბრაუზერში. ლექციაზე მისი გააქტიურება აგენტური შესაძლებლობების განბლოკვისთვის იყო საჭირო, რაც AI-ს ინტერნეტთან ურთიერთქმედების საშუალებას აძლევს.
