# Multi-Agent Architecture

> ქართულად: **მულტი-აგენტური არქიტექტურა**

## აღწერა

მულტი-აგენტური არქიტექტურა გულისხმობს რამდენიმე AI აგენტის პარალელურ მუშაობას, სადაც თითოეული სხვადასხვა ქვეამოცანაზე ფოკუსირდება. Claude Code-ის 'Parallel Agent' ბრძანება ამ პრინციპს ახორციელებს, თუმცა LangGraph, CrewAI ან AutoGen-ის მსგავსი ფორმალური ჩარჩოები კურსში სისტემატურად არ განიხილება.
