# Computer Use Mode

> ქართულად: **კომპიუტერის მართვის რეჟიმი**

## აღწერა

AI აგენტის შესაძლებლობა, მართოს ბრაუზერი ან კომპიუტერი ისე, როგორც ამას ადამიანი გააკეთებდა. ეს გულისხმობს ვებ-გვერდებზე ნავიგაციას, ფორმების შევსებას, ინფორმაციის მოძიებას და სხვა ინტერაქციებს. Claude-ის ბრაუზერის გაფართოებას შეუძლია მომხმარებლის მოქმედებების ჩაწერა და მათი გამეორება ან დაგეგმვა. მიუხედავად იმისა, რომ ეს ფუნქცია ჯერ კიდევ დაუხვეწავია, ის სწრაფად ვითარდება და მომავალში მნიშვნელოვნად გაამარტივებს ციფრული ამოცანების ავტომატიზაციას.
