# CLI "code" Command

> ქართულად: **"code" ბრძანებათა ხაზის ინსტრუმენტი**

## აღწერა

"code" ბრძანებათა ხაზის ინსტრუმენტი არის უტილიტა, რომელიც საშუალებას აძლევს მომხმარებლებს, ტერმინალიდან გაუშვან VS Code და მართონ პროექტები. ლექციის მსვლელობისას მოხდა მისი ინსტალაცია Claude-ის დესკტოპ-აპლიკაციის კონფიგურაციის ფარგლებში, რაც აუცილებელია AI და IDE-ის ინტეგრირებული მუშაობისთვის.
