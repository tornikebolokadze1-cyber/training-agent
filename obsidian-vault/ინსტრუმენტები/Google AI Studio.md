# Google AI Studio

> ქართულად: **გუგლის AI სტუდიო**

## აღწერა

Google-ის ვებ-პლატფორმა Gemini მოდელების ტესტირებისა და პროტოტიპირებისთვის, რომელიც მოიცავს ჩატს, სურათების გენერაციას, ვიდეოს, მუსიკას, ხმის ტრანსკრიფციას და მინი-აპლიკაციების Build ფუნქციას. ნაჩვენები იყო, როგორც სრულყოფილი სატესტო გარემო Gemini-ს ხედვის (vision) შესაძლებლობების შესამოწმებლად პროდაქშენში ინტეგრაციამდე.

## პრაქტიკული გამოყენება

- Gemini-ს Build ფუნქციის გამოყენება მინი-აპლიკაციების პროტოტიპირებისთვის — კონკრეტული ფუნქციების არჩევა (სურათის ანალიზი, ტრანსკრიფცია, TTS) სწრაფი ვალიდაციისთვის. (მარტის ჯგუფი #1, ლექცია 10)
- ინტერიერის დიზაინის პროტოტიპი — მომხმარებელი ტვირთავს ბინის ფოტოს და იღებს სკანდინავიური, ინდუსტრიული ან მოდერნი სტილის სარემონტო დიზაინს. (მარტის ჯგუფი #1, ლექცია 4)
- API გასაღების გენერირება და Claude Code-სთან ინტეგრაცია, სადაც Free tier და Tier 2 Postpay დონეები სხვადასხვა გამოყენების მოთხოვნებს შეესაბამება. (მარტის ჯგუფი #1, ლექცია 5)
- Gemini API-ის ბილინგის პრობლემა გადაიჭრა ახალი API გასაღების ცალკე ელ-ფოსტით შექმნით და $300-იანი Google Cloud კრედიტების გამოყენების განმარტებით (მარტის ჯგუფი #1, ლექცია 6)
- პოდკასტის სტილის ქართული დიალოგი ქალი და მამაკაცი სპიკერებს შორის AI აგენტების თემაზე — Multi-speaker TTS-ის პრაქტიკული გამოყენება. (მარტის ჯგუფი #1, ლექცია 7)
- კაფეს სარეკლამო სურათი დაგენერირდა ქართული ტექსტით 'საქალაქო საუკეთესო ყავა' — უნაკლო მართლწერით და 4K რეზოლუციით. (მარტის ჯგუფი #1, ლექცია 8)
- Gemini API გასაღების შექმნა და პრაქტიკულად დემონსტრირება, სურათების, ვიდეოსა და TTS გენერაციის ფუნქციების ჩვენება. (მარტის ჯგუფი #2, ლექცია 5)
- მრავალმოლაპარაკიანი დიალოგის გენერაცია ქოლ-ცენტრის სცენარით — Speaker 1 გაბრაზებული მომხმარებელი, Speaker 2 ოპერატორი. (მარტის ჯგუფი #2, ლექცია 7)
