# GitHub

## აღწერა

ღრუბელზე დაფუძნებული პლატფორმა, რომელიც მასპინძლობს Git რეპოზიტორიებს, რაც საშუალებას იძლევა კოდის ვერსიების მართვის, სარეზერვო ასლების შექმნისა და გუნდური თანამშრომლობისთვის. ის გამოიყენება როგორც პირადი, ისე საჯარო (open source) პროექტებისთვის, სადაც დეველოპერები იზიარებენ და აუმჯობესებენ კოდს.
