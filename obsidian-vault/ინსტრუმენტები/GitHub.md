# GitHub

## აღწერა

GitHub არის ვებ-ზე დაფუძნებული პლატფორმა, რომელიც Git-ის ვერსიების კონტროლის სისტემის გამოყენებით უზრუნველყოფს კოდის ჰოსტინგს და პროგრამული უზრუნველყოფის ერთობლივ განვითარებას. ის წარმოადგენს დისტანციურ სარეზერვო რეპოზიტორს, სადაც პროექტის ასლები უსაფრთხოებისთვის ინახება და გუნდებს საშუალებას აძლევს, ითანამშრომლონ კოდზე და გაუზიარონ ცვლილებები ერთმანეთს.
