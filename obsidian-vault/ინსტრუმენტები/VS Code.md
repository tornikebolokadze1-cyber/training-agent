# VS Code

## აღწერა

VS Code არის Microsoft-ის მიერ შექმნილი უფასო, ღია კოდის კოდის რედაქტორი, რომელიც მხარს უჭერს სხვადასხვა პროგრამირების ენას და გაფართოებას. ლექციაზე ის წარმოადგენდა ძირითად ინტეგრირებულ განვითარების გარემოს (IDE), რომელშიც Claude AI-ის ინტეგრაცია და პრაქტიკული გამოყენება დემონსტრირდა კოდირების ამოცანებისთვის.
