# VS Code

## აღწერა

Visual Studio Code (VS Code) არის უფასო და ღია კოდის ინტეგრირებული განვითარების გარემო (IDE), რომელიც აერთიანებს კოდის რედაქტორს, დებაგერს, Git-ის ინტეგრაციას და ტერმინალს. ის ფართოდ გამოიყენება პროგრამისტების მიერ და წარმოადგენს მძლავრ პლატფორმას Claude Code-თან ინტეგრაციისთვის, ექსტენსიების საშუალებით მისი ფუნქციონალურობის გასაფართოებლად.
