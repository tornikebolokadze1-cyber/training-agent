# Git

## აღწერა

Git არის განაწილებული ვერსიების კონტროლის სისტემა, რომელიც გამოიყენება ცვლილებების თვალყურის დევნებისთვის ნებისმიერი ტიპის ფაილებში. ის საშუალებას აძლევს მომხმარებლებს, ლოკალურად შეინახონ პროექტის ასლები და მართონ ვერსიები პირად სამუშაო სივრცეში, რაც უზრუნველყოფს კოდის ისტორიის შენარჩუნებას და ცვლილებების უკუგდების შესაძლებლობას.
