# Git

## აღწერა

ეს არის ლოკალური ვერსიების კონტროლის სისტემა, რომელიც საშუალებას აძლევს მომხმარებლებს აკონტროლონ ცვლილებები ფაილებში და შეინახონ თავიანთი სამუშაოს სხვადასხვა ვერსია (commits). Git არის GitHub-ის საფუძველი და აუცილებელია პროექტის ისტორიის მართვისა და ცვლილებებზე თვალყურის დევნებისთვის.
