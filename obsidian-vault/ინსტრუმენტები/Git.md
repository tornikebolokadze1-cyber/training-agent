# Git

> ქართულად: **გიტი**

## აღწერა

კოდის ვერსიების კონტროლის სისტემა, რომელიც საშუალებას იძლევა შენახული იქნეს (commit) და სინქრონიზებული GitHub-თან. ინსტრუქტორმა რეალური გამაფრთხილებელი მაგალითი მოიყვანა — ვიღაცამ დაკარგა კოდის ცვლილებები, რადგან ისინი Git-ში არ იყო შენახული.
