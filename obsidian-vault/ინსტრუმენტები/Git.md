# Git

> ქართულად: **გიტი**

## აღწერა

კოდის ვერსიების კონტროლის სისტემა, რომელიც საშუალებას იძლევა ცვლილებების შენახვა (commit) და ტრეკინგი. ინსტრუქტორმა ხაზი გაუსვა Git-ში შენახვის კრიტიკულ მნიშვნელობას — ვიღაცამ დაკარგა მთელი სამუშაო, რადგან კოდი committed არ ყოფილა.
