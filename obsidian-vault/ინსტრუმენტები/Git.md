# Git

> ქართულად: **გიტი**

## აღწერა

Git არის ვერსიების კონტროლის სისტემა, რომელიც კოდისთვის Google Docs-ის ვერსიების ისტორიის ანალოგია. AI აგენტების განვითარების პროცესში Git-ი აუცილებელი ინსტრუმენტია, რათა ცვლილებების ისტორია შენახულ იქნეს და შეცდომების შემთხვევაში წინა ვერსიაზე დაბრუნება შესაძლებელი იყოს.
