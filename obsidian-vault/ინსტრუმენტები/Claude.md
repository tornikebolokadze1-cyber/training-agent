# Claude

> ქართულად: **კლოდი**

## აღწერა

Anthropic-ის მიერ შემუშავებული LLM, რომელიც ლექციაში განიხილება როგორც ძირითადი ინსტრუმენტი AI აგენტების შენებისთვის, VS Code-ში Claude Code-ის სახით. მოიცავს Skills ფუნქციას სპეციფიკური დომენების სწავლებისთვის, auto-compact ფუნქციას კონტექსტის მართვისთვის და მეხსიერების სისტემას სესიებს შორის ინფორმაციის შენარჩუნებისთვის.

## პრაქტიკული გამოყენება

- N8N workflow-ების ავტომატური გენერაცია VS Code-ში Codex-ის მეშვეობით — ექსპერტებიც კი ამ მიდგომას ხელით კეთებას ამჯობინებენ. (მარტის ჯგუფი #1, ლექცია 10)
- Facebook Ads-ის ყოველკვირეული შედეგების შეჯამება AI აგენტის გამოყენებით — ციფრული მარკეტინგის სპეციალისტ ლიკას რეალური პროექტი, სადაც Claude Chat-ში ხდება აგენტის იდეაციის პირველი ეტაპი. (მარტის ჯგუფი #1, ლექცია 15)
- მონაცემთა ბაზის ვარიანტების კვლევა: Claude-ს ეთხოვება ინტერნეტში მოძებნოს 2026 წლის მაისის მდგომარეობით მონაცემთა ბაზების ვარიანტები და დაბრუნოს 3-5 საუკეთესო ვარიანტი დადებითი და უარყოფითი მხარეებით. (მარტის ჯგუფი #1, ლექცია 15)
- საქართველოს ეროვნული ბანკის ვებსაიტზე Python სკრიპტის ლაივ-რეჟიმში დაწერა, მონაცემების ამოღება და ეკონომიკური ანალიზის გენერირება (მარტის ჯგუფი #1, ლექცია 2)
- Claude Design-ით LinkedIn-ისთვის 2 წ. 22 წამიანი ანიმირებული დიზაინის ვიდეოს და კარუსელის შექმნა სკრიპტისა და ავატარის ვიდეოსგან. (მარტის ჯგუფი #2, ლექცია 12)
- პირდაპირ დესკტოპიდან ნარინჯისფერი აიკონით Claude-ის გახსნა და ფლაგინების სექციის მიმოხილვა. (მარტის ჯგუფი #2, ლექცია 14)
- Canva-ს კონექტორის გამოყენებით Claude-მა პირდაპირ Canva-ში ავტომატურად დააგენერირა პრეზენტაცია AI აგენტების თემაზე. (მარტის ჯგუფი #2, ლექცია 3)
- სტუდენტ მიშას GitHub პროექტი Claude-ს გაუგზავნეს შეფასებისთვის; მოდელმა კოდს, უსაფრთხოებას, Git-ს და CI/CD-ს მისცა ქულები და დეტალური კრიტიკა. (მარტის ჯგუფი #2, ლექცია 3)
- მარიამის Meta Ads-ის აგენტი: Claude Skills-ის გამოყენება რეკლამის ტექსტების პრიორიტეტიზაციისა და გენერაციისთვის, ვექტორული ბაზა კონკურენტების სარეკლამო შინაარსის სემანტიკური ანალიზისთვის. (მარტის ჯგუფი #2, ლექცია 6)
- VS Code-ში ბუღალტერიის განყოფილებისთვის ემეილ რეპორტების ავტომატური გაგზავნა განრიგის მიხედვით, WhatsApp მასობრივი შეტყობინებების სისტემა და B2B CRM ვიტამინების დისტრიბუტორისთვის. (მარტის ჯგუფი #2, ლექცია 9)
- Facebook/LinkedIn-ის პოსტინგ-აგენტი, რომელიც Telegram-ში უგზავნის პოსტის პროექტებს 'დაამტკიცე / უარყავი / ხელახლა გენერირება' ღილაკებით გამოქვეყნებამდე დამტკიცებისთვის. (მაისის ჯგუფი #1, ლექცია 1)
