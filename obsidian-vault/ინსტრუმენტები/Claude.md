# Claude

## აღწერა

Anthropic-ის LLM, რომელიც ლექციაზე გამოიყენებოდა VS Code-ში Codex-ის ინტეგრაციით N8N workflow-ების გენერაციისა და პროექტების განვითარებისთვის. ნაჩვენები იყო, რომ VS Code-ში Claude-დან სხვა მოდელზე გადასვლისას ახალ მოდელს პირდაპირ უნდა ეთქვას, რომ პროექტი Claude-ის მიერ იყო აშენებული.

## პრაქტიკული გამოყენება

- N8N workflow-ების ავტომატური გენერაცია VS Code-ში Codex-ის მეშვეობით — ექსპერტებიც კი ამ მიდგომას ხელით კეთებას ამჯობინებენ. (ჯგუფი 1, ლექცია 10)
- ლიკა გამოიყენებს Claude-ს WhatsApp-ით მიღებული კოდის ანალიზისა და გამოსწორებისთვის, პრომპტების მიწოდებით, რომ Claude ერთი ხელი გამეორებითი შეცდომებში აღმოჩნდება (ჯგუფი 1, ლექცია 13)
- Facebook Ads-ის ყოველკვირეული შედეგების შეჯამება AI აგენტის გამოყენებით — ციფრული მარკეტინგის სპეციალისტ ლიკას რეალური პროექტი, სადაც Claude Chat-ში ხდება აგენტის იდეაციის პირველი ეტაპი. (ჯგუფი 1, ლექცია 15)
- მონაცემთა ბაზის ვარიანტების კვლევა: Claude-ს ეთხოვება ინტერნეტში მოძებნოს 2026 წლის მაისის მდგომარეობით მონაცემთა ბაზების ვარიანტები და დაბრუნოს 3-5 საუკეთესო ვარიანტი დადებითი და უარყოფითი მხარეებით. (ჯგუფი 1, ლექცია 15)
- საქართველოს ეროვნული ბანკის ვებსაიტზე Python სკრიპტის ლაივ-რეჟიმში დაწერა, მონაცემების ამოღება და ეკონომიკური ანალიზის გენერირება (ჯგუფი 1, ლექცია 2)
- Python სკრიპტების წერა დამოუკიდებელი ვებ-ბრაუზეროვანი მოქმედებებისთვის (ჯგუფი 2, ლექცია 3)
- ბიზნეს აგენტის შენება პერსონალური პროექტების მართვის რთული სისტემით, რომელიც მოიცავს წესებს, უნარებს და სესიებს შორის მეხსიერებას (ჯგუფი 2, ლექცია 9)
