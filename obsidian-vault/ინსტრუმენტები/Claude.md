# Claude

> ქართულად: **კლოდი**

## აღწერა

Anthropic-ის მიერ შემუშავებული LLM, რომელიც ლექციაში განიხილება როგორც ძირითადი ინსტრუმენტი AI აგენტების შენებისთვის, VS Code-ში Claude Code-ის სახით. მოიცავს Skills ფუნქციას სპეციფიკური დომენების სწავლებისთვის, auto-compact ფუნქციას კონტექსტის მართვისთვის და მეხსიერების სისტემას სესიებს შორის ინფორმაციის შენარჩუნებისთვის.

## პრაქტიკული გამოყენება

- N8N workflow-ების ავტომატური გენერაცია VS Code-ში Codex-ის მეშვეობით — ექსპერტებიც კი ამ მიდგომას ხელით კეთებას ამჯობინებენ. (ჯგუფი 1, ლექცია 10)
- ლიკა გამოიყენებს Claude-ს WhatsApp-ით მიღებული კოდის ანალიზისა და გამოსწორებისთვის, პრომპტების მიწოდებით, რომ Claude ერთი ხელი გამეორებითი შეცდომებში აღმოჩნდება (ჯგუფი 1, ლექცია 13)
- Facebook Ads-ის ყოველკვირეული შედეგების შეჯამება AI აგენტის გამოყენებით — ციფრული მარკეტინგის სპეციალისტ ლიკას რეალური პროექტი, სადაც Claude Chat-ში ხდება აგენტის იდეაციის პირველი ეტაპი. (ჯგუფი 1, ლექცია 15)
- მონაცემთა ბაზის ვარიანტების კვლევა: Claude-ს ეთხოვება ინტერნეტში მოძებნოს 2026 წლის მაისის მდგომარეობით მონაცემთა ბაზების ვარიანტები და დაბრუნოს 3-5 საუკეთესო ვარიანტი დადებითი და უარყოფითი მხარეებით. (ჯგუფი 1, ლექცია 15)
- საქართველოს ეროვნული ბანკის ვებსაიტზე Python სკრიპტის ლაივ-რეჟიმში დაწერა, მონაცემების ამოღება და ეკონომიკური ანალიზის გენერირება (ჯგუფი 1, ლექცია 2)
- Claude Design-ით LinkedIn-ისთვის 2 წ. 22 წამიანი ანიმირებული დიზაინის ვიდეოს და კარუსელის შექმნა სკრიპტისა და ავატარის ვიდეოსგან. (ჯგუფი 2, ლექცია 12)
- პირდაპირ დესკტოპიდან ნარინჯისფერი აიკონით Claude-ის გახსნა და ფლაგინების სექციის მიმოხილვა. (ჯგუფი 2, ლექცია 14)
- Canva-ს კონექტორის გამოყენებით Claude-მა პირდაპირ Canva-ში ავტომატურად დააგენერირა პრეზენტაცია AI აგენტების თემაზე. (ჯგუფი 2, ლექცია 3)
- სტუდენტ მიშას GitHub პროექტი Claude-ს გაუგზავნეს შეფასებისთვის; მოდელმა კოდს, უსაფრთხოებას, Git-ს და CI/CD-ს მისცა ქულები და დეტალური კრიტიკა. (ჯგუფი 2, ლექცია 3)
- მარიამის Meta Ads-ის აგენტი: Claude Skills-ის გამოყენება რეკლამის ტექსტების პრიორიტეტიზაციისა და გენერაციისთვის, ვექტორული ბაზა კონკურენტების სარეკლამო შინაარსის სემანტიკური ანალიზისთვის. (ჯგუფი 2, ლექცია 6)
- VS Code-ში ბუღალტერიის განყოფილებისთვის ემეილ რეპორტების ავტომატური გაგზავნა განრიგის მიხედვით, WhatsApp მასობრივი შეტყობინებების სისტემა და B2B CRM ვიტამინების დისტრიბუტორისთვის. (ჯგუფი 2, ლექცია 9)
