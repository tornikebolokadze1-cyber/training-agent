# Google Antigravity / Project IDX

## აღწერა

Google-ის ბრაუზერზე დაფუძნებული IDE, წარმოდგენილი VS Code-ის ალტერნატივად, განსაკუთრებით Front-end/UI გენერირებისთვის. ლექტორმა რეკომენდაცია გაუწია მას, როგორც საწყის პლატფორმას დამწყებთათვის, Gemini-სთან ერთად უფასო გამოყენების ლიმიტებისა და VS Code-თან შედარებით დაბალი სირთულის გამო.
