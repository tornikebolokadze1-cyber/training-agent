# Claude Computer Use

> ქართულად: **კომპიუტერის მართვა Claude-ით**

## აღწერა

Claude-ის ახალი შესაძლებლობა, რომელიც AI-ს საშუალებას აძლევს ავტომატურად მართოს კომპიუტერი — გახსნას ფაილები, იბრაუზეროს ინტერნეტში, შეავსოს ფორმები და გაუშვას კოდი. ხელმისაწვდომია Claude Pro და Max გეგმებზე და ირთვება Claude Desktop App-ის პარამეტრებიდან.
