# Microsoft Graph API

## აღწერა

Microsoft-ის API, რომელიც წვდომას უზრუნველყოფს Microsoft 365 სერვისების მონაცემებზე, მათ შორის Teams-ის შეხვედრებისა და ჩანაწერებზე. თინათინის პროექტში გამოიყენებოდა ასინქრონული ტრანსკრიფციისთვის — სისტემა Teams-ის ჩანაწერებს Graph API-ის მეშვეობით ამუშავებს შეხვედრის დასრულების შემდეგ.
