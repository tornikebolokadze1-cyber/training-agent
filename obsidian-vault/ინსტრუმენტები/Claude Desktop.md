# Claude Desktop

## აღწერა

ეს არის Anthropic-ის მიერ შექმნილი დესკტოპ აპლიკაცია, რომელიც მნიშვნელოვნად უფრო ძლიერია, ვიდრე Claude-ის ვებ ინტერფეისი და იძლევა კომპიუტერის ფაილებსა და პროცესებზე წვდომის საშუალებას. ის მოიცავს სამ ძირითად ინტერფეისს: Chat, Code და Cowork, რომელთაგან თითოეული განსხვავებულ ფუნქციონალს გვთავაზობს მომხმარებლის საჭიროებების მიხედვით.

- სამივე ტაბის (Chat, Cowork, Code) ჩვენება და დაგეგმილი დავალებების დემო (კალენდრის შეჯამება, ევროსასამართლოს სიახლეები). (ჯგუფი 2, ლექცია 2)
- დაგეგმილი დავალებების შექმნა (სახელით, აღწერით, პრომპტით, სიხშირით). (ჯგუფი 2, ლექცია 2)
- საქართველოს შრომის ბაზრის კვლევა, ხელფასების დადგენა, IT მენეჯერის ვაკანსიის 3 ვარიანტის შექმნა Word-ში, LinkedIn-ზე ტოპ 10 კანდიდატის მოძიება Playwright CLI-ით, PowerPoint პრეზენტაციის შექმნა და Excel-ში შედარებითი ანალიზი. (ჯგუფი 1, ლექცია 3)
- Claude Skills ფუნქციის გამოყენებით ლექციის რეპორტის ავტომატური გენერაციის უნარის შექმნა ინტერაქტიული კითხვა-პასუხის რეჟიმში. (ჯგუფი 1, ლექცია 2)
