# N8N

> ქართულად: **ენ8ენ**

## აღწერა

ღია კოდის სამუშაო პროცესების (workflow) ავტომატიზაციის პლატფორმა, რომელიც საშუალებას იძლევა კოდის გარეშე ან მცირე კოდით შეიქმნას რთული ინტეგრაციები. ლექციაზე ხაზი გაესვა, რომ გლობალური N8N ექსპერტებიც კი ახლა იყენებენ Claude-ს workflow-ების ასაწყობად, ამიტომ ხელით ყველაფრის კეთება მოძველებული პრაქტიკაა.

## პრაქტიკული გამოყენება

- მაკას გალერეის ბიზნესისთვის მარტივი, განმეორებადი ამოცანების ავტომატიზაცია — ყოველდღიური სოციალური მედიის პოსტები და ელ-ფოსტის კამპანიები. (ჯგუფი 1, ლექცია 12)
- Claude Code-სა და N8N-ს შორის MCP სერვერის კონფიგურაციის მეშვეობით ინტეგრაციის დემონსტრაცია, მათ შორის სამუშაო პროცესის მართვის ბრძანებების კონფიგურაციის ჩათვლით. (ჯგუფი 1, ლექცია 14)
- სამუშაო პროცესების ავტომატიზაცია ლოკალური (უფასო) ვერსიით — ლექტორი ურჩევს ამ ეტაპზე $28/თვე ქლაუდ ვერსიაში ინვესტიციის გადავადებას და ამ ბიუჯეტის Claude/API წვდომაში გადამისამართებას. (ჯგუფი 1, ლექცია 15)
- Telegram-ზე მომხმარებელთა გამოხმაურებების მონიტორინგი, კონკურენტების ფასების ავტომატური თვალყურის დევნება და გაყიდვის პროცესის სრული ავტომატიზაცია (ჯგუფი 1, ლექცია 2)
- ვიზუალური AI Agent workflow-ის ასაწყობად, სადაც AI Agent კვანძი დაკავშირებულია კალკულატორთან, Slack-თან, Google Sheets-თან, Gmail-სა და Pinecone Vector Store-თან. (ჯგუფი 1, ლექცია 3)
- ანიტამ Claude Code-ის გამოყენებით N8N ავტომატიზაციის ფაიფლაინი ააწყო; თორნიკემ რეკომენდაცია გასცა, Claude Code-ი N8N-ის ნაცვლად გამოიყენოს სადაც შესაძლებელია. (ჯგუფი 2, ლექცია 12)
- Telegram შეტყობინებების მიღება, ლოკალური LLM-ის გამოძახება და ავტომატური პასუხის გაგზავნა — ნულოვანი ოპერაციული ხარჯებით (ჯგუფი 2, ლექცია 13)
- სამუშაო პროცესების ავტომატიზაციისთვის გამოყენება; სესიაზე გადამოწმდა, ჰქონდათ თუ არა მონაწილეებს ინსტრუმენტი დაინსტალირებული. (ჯგუფი 2, ლექცია 14)
- N8N MCP-ის მეშვეობით Claude Code-ს შეუძლია ააგოს ავტომატიზაციის სამუშაო პროცესები პირდაპირ N8N-ის ვიზუალურ გარემოში ნედლი კოდის წერის გარეშე. (ჯგუფი 2, ლექცია 2)
- Claude Desktop Dispatch-თან ინტეგრაციით დაგეგმილი ავტომატური ამოცანების კონფიგურაცია, მაგალითად ყოველდღიური ანგარიშების გენერირება და გაგზავნა. (ჯგუფი 2, ლექცია 4)
- სოციალური მედიის პოსტების ავტომატური გენერაცია და Facebook-ზე გამოქვეყნება განრიგის მიხედვით Claude-ის გამოყენებით. (ჯგუფი 2, ლექცია 9)
