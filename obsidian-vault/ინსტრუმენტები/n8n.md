# N8N

## აღწერა

n8n არის სამუშაო პროცესების ავტომატიზაციის ღია კოდის პლატფორმა, რომელიც მომხმარებლებს საშუალებას აძლევს, დააკავშირონ სხვადასხვა სერვისი და შექმნან AI აგენტები კვანძებზე დაფუძნებული ვიზუალური ინტერფეისის გამოყენებით. ის ხელმისაწვდომია როგორც ღრუბლოვანი (ფასიანი), ასევე ლოკალური (უფასო) თვითმასპინძლობის ვერსიებში, რაც მონაცემთა კონფიდენციალურობას უზრუნველყოფს.

## პრაქტიკული გამოყენება

- შორენამ აწყობს ავტომატიზებულ სამუშაო ნაკადს Claude Code-ის MCP ინტეგრაციით N8N ლოკალური პარამეტრებით (ჯგუფი 1, ლექცია 12)
- N8N-ის ინტეგრირება Claude Code-თან MCP სერვერის კონფიგურაციის მეშვეობით, რაც საშუმებას იძლევა AI-ს მართოს ავტომატიზირებული სამუშაო პროცესები (ჯგუფი 1, ლექცია 14)
- სამუშაო პროცესების ავტომატიზაცია ლოკალური (უფასო) ვერსიით — ლექტორი ურჩევს ამ ეტაპზე $28/თვე ქლაუდ ვერსიაში ინვესტიციის გადავადებას და ამ ბიუჯეტის Claude/API წვდომაში გადამისამართებას. (ჯგუფი 1, ლექცია 15)
- Telegram-ზე მომხმარებელთა გამოხმაურებების მონიტორინგი, კონკურენტების ფასების ავტომატური თვალყურის დევნება და გაყიდვის პროცესის სრული ავტომატიზაცია (ჯგუფი 1, ლექცია 2)
- ვიზუალური AI Agent workflow-ის ასაწყობად, სადაც AI Agent კვანძი დაკავშირებულია კალკულატორთან, Slack-თან, Google Sheets-თან, Gmail-სა და Pinecone Vector Store-თან. (ჯგუფი 1, ლექცია 3)
- რთული სამუშაო პროცესების ავტომატიზაცია ვიზუალური კვანძებითა და Claude Code ინტეგრაციით, ნედლი კოდის წერის გარეშე (ჯგუფი 2, ლექცია 2)
- Facebook ჯგუფების სკრეპინგი, ინფორმაციის აგრეგაცია და მაჯიელე შეტყობინებების გამოგზავნა (ჯგუფი 2, ლექცია 9)
