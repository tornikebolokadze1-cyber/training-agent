# Telegram Bot

> ქართულად: **ტელეგრამ ბოთი**

## აღწერა

Telegram მესინჯერის API-ზე დაფუძნებული ავტომატიზებული სისტემა, რომელიც სხვადასხვა ბიზნეს-ამოცანების — დავალებების რეგისტრაციის, შეტყობინებების გაგზავნის, სოციალური მედიის მართვის — ავტომატიზაციისთვის გამოიყენება. სტუდენტთა პროექტებში ყველაზე გავრცელებული ინტეგრაციის წერტილია.

## პრაქტიკული გამოყენება

- მონაწილე მიშას არსებული Telegram ბოტი, რომელიც ბლოგპოსტებს წერს და ვებსაიტის ანალიტიკას აგზავნის — განხილულ იქნა PaperClip-ში ინტეგრაციის კონტექსტში. (მარტის ჯგუფი #2, ლექცია 14)
- ნიკომ შექმნა იუმორისტული ბოტი მეგობრებისთვის; ლოიალობის პროგრამის აპლიკაცია და კომპანიის შიდა პორტალი. (მარტის ჯგუფი #2, ლექცია 5)
- BotFather-ის მეშვეობით ელექტრონული კომერციის ჩატბოტის (@E-commerce_Agent_ChatBot_2027) შექმნა და API ტოკენის მიღება Notion-სა და AirTable-თთან ინტეგრაციისთვის (მაისის ჯგუფი #1, ლექცია 2)
- ClickUp-ში ავტომატური დავალებების რეგისტრაცია Telegram ბოტის საშუალებით, Railway სერვერზე გაშვებით (მაისის ჯგუფი #1, ლექცია 3)
- ელ-კომერციის კლიენტების მხარდაჭერის ბოტი: BotFather-ის მეშვეობით შეიქმნა 'E-commerce Georgian Agent Bot', რომელიც Airtable-სა და Notion-თან ინტეგრირებული AI აგენტის მომხმარებელთა ინტერფეისს წარმოადგენს. (მაისის ჯგუფი #2, ლექცია 1)
