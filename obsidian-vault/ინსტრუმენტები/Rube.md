# Rube

## აღწერა

MCP (Model Context Protocol) სერვერის დემონსტრაციისთვის გამოყენებული პლატფორმა. ლექციაზე ნაჩვენები იყო, თუ როგორ დაუკავშირდა Rube YouTube Analytics-სა და Gmail-ს და წარმატებით გააანალიზა YouTube არხის სტატისტიკა, შემდეგ კი მოამზადა და გაგზავნა რეპორტი ელ. ფოსტაზე. ეს დემო აჩვენებს MCP სერვერების პრაქტიკულ გამოყენებას მრავალი აპლიკაციის ავტომატიზებული ინტეგრაციისთვის.
