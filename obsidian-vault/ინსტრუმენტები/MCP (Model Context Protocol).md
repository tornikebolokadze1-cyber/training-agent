# MCP (Model Context Protocol)

> ქართულად: **Model Context Protocol (მოდელის კონტექსტის პროტოკოლი)**

## აღწერა

ფუნდამენტური კავშირის პროტოკოლი, რომელიც აკავშირებს ბევრ აპლიკაციას ერთ სერვერთან (USB ჰაბის ანალოგია). ეს ერთი წერტილი მართავს ყველა ინტეგრაციას Claude-ის (ან სხვა AI-ის) ბევრ გარე სერვისთან. აიხსნა როგორც USB ჰაბი, რომელიც აკავშირებს ბევრ აპლიკაციას ერთ კავშირად.
