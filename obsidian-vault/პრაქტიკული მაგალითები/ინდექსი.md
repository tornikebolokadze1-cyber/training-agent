# პრაქტიკული მაგალითები

## ჯგუფი 1, ლექცია 13

- [[Claude AI / Claude Desktop Application]] -- WhatsApp-ით მიღებული კოდის გადატანა Claude-ის სესიაში და პრომპტებთან ერთად მიწოდება, რათა მოდელმა შეასრულოს კონკრეტული დავალება.
- [[Claude Desktop Application]] -- სამაგიდო აპლიკაციის პარამეტრებში ნავიგაცია და აგენტური შესაძლებლობების (მაგ., "computer use" და "browser action" ნებართვები) ჩართვა, რაც AI-ს საშუალებას აძლევს ურთიერთქმედებდეს კომპიუტერსა და ბრაუზერთან.
- [[VS Code]] -- ახალი ლოკალური საქაღალდის შექმნა პროექტის ორგანიზებისთვის კოდის გაშვებამდე, რაც კოდირების საუკეთესო პრაქტიკის ნაწილია.

- [[VS Code + Claude Code]] -- HR აგენტის სრული აწყობა ნაბიჯ-ნაბიჯ სპეციალური ბრძანებებით (/init-project, /create-prd, /architecture-review, /generate-tests, /implement, /commit). აგენტის ფუნქციები მოიცავდა Gmail-ის მონიტორინგს, Telegram ნოტიფიკაციებს, ვექტორულ ბაზას, Enagram AI-ით ინტერვიუს ტრანსკრიფციას და LinkedIn-დან კანდიდატების სკრეპინგს.
- [[Claude Desktop (Cowork mode) + Anything CLI]] -- Claude Code-ის ეკოსისტემის ინფოგრაფიკის შექმნა Draw.io-ში ერთი წინადადების გამოყენებით.
- [[Google Antigravity IDE]] -- Gemini-ს მიერ წინა სესიაზე შექმნილი ქართული სამზარეულოს ვებსაიტის ჩვენება (AI გენერირებული სურათებით), მიუხედავად დემოს გათიშვისა.

## ჯგუფი 1, ლექცია 7

- [[Cartesia]] -- ლოპოტას რეზორტის ტექსტის წაკითხვა, ტექსტის ფორმატირების (რიცხვების სიტყვიერად დაწერა, მძიმეების ნაცვლად წერტილების გამოყენება პაუზისთვის) მნიშვნელობის დემონსტრირება, ხმის კლონირება და ლოკალიზაცია (ქართულიდან ესპანურზე).
- [[ElevenLabs]] -- იგივე ლოპოტას ტექსტის წაკითხვა (AI-მ რიცხვები თავისით სწორად წაიკითხა), ხმის ეფექტების (SFX) გენერაცია (დაცემინების ხმა სპეციფიკური პარამეტრებით) და ვიდეოს დუბლირების ფუნქცია.
- [[Google AI Studio]] -- პოდკასტის სტილის დიალოგი ქართულ ენაზე ორ სპიკერს (ქალი და მამაკაცი) შორის AI აგენტების შესახებ, რაც მრავალმომხსენებლიან მხარდაჭერას აჩვენებს.
- [[Rask AI]] -- სამოგზაურო ვიდეოს ინგლისურიდან ქართულად თარგმნა და გახმოვანება, Lipsyncing-ის ფუნქციის ახსნა და მისი მაღალი ღირებულების ხაზგასმა.
- [[Gemini/Lyria]] -- ქართული ლექსის მუსიკად ქცევა, ალბომის ყდის ავტომატურად შექმნა, თუმცა Suno-სთან შედარებით დაბალი ხარისხით.
- [[Suno]] -- იგივე ქართული ლექსის მუსიკად ქცევა ბევრად უკეთესი ხარისხით, მარტივი და რთული (Advanced) რეჟიმების დემონსტრაცია.
